YL’s Clothing – E-commerce UI Design

Design · UI/UX Figma Desktop & Mobile

YL’s Clothing is a high-fidelity e-commerce website concept that focuses on a clean, modern, and easy-to-browse shopping experience for sustainable fashion. Both desktop and mobile layouts were designed for a seamless, consistent look.

View Figma Prototype

Overview

The goal of this project was to design a shopping experience that feels clean, trustworthy, and effortless. The target users are students and young professionals who value simplicity, clarity, and fast interactions.

The UI emphasises strong hierarchy, modern typography, spaced layouts, and meaningful call-to-action buttons for smooth navigation.

Project Details

  • Role: End-to-end UI/UX Designer
  • Timeline: ~2 weeks (school project)
  • Platform: Desktop & Mobile Web
  • Tools: Figma, custom colour palette

Design Process

From initial direction to polished high-fidelity screens.

01 · Discover & Define

Studied top clothing websites, identified best UX patterns, and defined goals: simple navigation, visible promotions, and smooth checkout experience.

02 · Wireframes

Built low-fidelity wireframes for homepage, product listing, product details, cart, and checkout. Focused purely on layout and clarity.

03 · Visual Design

Added final colours, spacing, imagery, reusable components, and responsive layouts. The design uses clean contrast and bold typography.

04 · Prototype

Created an interactive Figma prototype showing navigation, hover states, and interaction flow. Useful for presenting the design concept clearly.
