Task Manager App

React Native Mobile UI State Management

The Task Manager App is a productivity app built using React Native. Users can add tasks, update their status, and delete tasks. It helped me strengthen my skills in component structure, state handling, and building efficient lists for mobile applications.

Watch Demo Video

* This narration video walks through the app features and key code sections.

Overview

This Task Manager App allows users to stay organised and keep track of their daily tasks. The app includes features to add new tasks, set the status (Not Started, In Progress, Completed), and remove tasks.

This project strengthened my understanding of controlled inputs, state management, conditional rendering, and designing clean UI layouts in React Native.

Project Details

  • Role:Developer
  • Type: React Native learning project
  • Tech Stack: React Native, JavaScript
  • Core Concepts: State, Lists, Props, UI Components

Key Features

Lightweight, functional, and mobile-friendly.

Add Tasks

A clean input form allows users to add tasks instantly. Data is stored in state and displayed automatically.

Update Status

Each task includes an option to change the status — Completed, In Progress, or Not Started.

Delete Tasks

Tasks can be deleted using a single tap. Built using React's immutable state updates.

What I Learned

This project helped me improve as a mobile developer. I became more confident in:

  • Managing React Native component structure efficiently
  • Using state and event handlers to update the UI.
  • Rendering dynamic lists using FlatList.
  • Designing clean mobile UIs with flexible layouts.

Next Steps
